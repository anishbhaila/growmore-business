== Description ==

Growmore Business is a versatile theme designed for any website, with templates for business, blogging, and portfolios. Easy adjustments to color and typography offer diverse possibilities. The theme streamlines site building with various styles, page layouts, and full compatibility with the WordPress 6.4 design tools.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
